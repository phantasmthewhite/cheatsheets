* -a 3 = brute-force
* -m 0 = hash MD5

### Straight (Salted MD5)
* -a 0 = straight attack
* -m 10 = salted MD5

Soit:

`hashcat -a 0 -m 10 hash.txt 1000000-password-seclists.txt`
